# sign-language-kiosk
> LSTM을 이용한 수어번역 모델

<br><br>

## 프로젝트 소개
> 햄버거(키오스크) 주문 상황을 가정하여 22개 단어를 선정하고,<br>
> 22개 단어에 대한 수어 영상을 직접 촬영하여 데이터셋을 제작했습니다.<br>
> 직접 구축한 데이터셋을 LSTM모델에 학습 시키고 웹캠을 통해 시연했습니다.
